# Модуль оператора лазерной резки

## Описание
Централизованный модуль для управления заявками на лазерную резку со всех участков производства (U2, U3, U4, U5).

## Доступ
Модуль доступен пользователям с ролями:
- `admin` - Администратор системы
- `director` - Директор  
- `laser_operator` - Оператор лазерной резки

## Функциональность
- Просмотр всех заявок на лазерную резку со всех участков
- Фильтрация заявок по статусу и участку
- Отметка заявок как выполненных
- Отображение времени выполнения

## Установка
1. Выполните SQL скрипт для добавления роли:
   ```sql
   source add_laser_operator_role.sql
   ```

2. Назначьте роль `laser_operator` нужным пользователям через админ-панель

## Структура файлов
