# План добавления статистики по ножам под таблицей

## Описание задачи

Добавить блок статистики под календарной таблицей, который показывает для каждого комплекта ножей:
- Количество дней в каждом статусе (в запасе, в заточке, в работе) за выбранный период
- Процент времени в каждом статусе
- Последний установленный статус и дата

## Файлы для изменения

1. `plan_U3/knives_bobinorezka.php`
2. `plan_U3/knives_prosechnik.php`

## Детали реализации

### 1. Расчет статистики (после получения данных календаря, строка ~122)

Добавить код расчета статистики после блока получения данных календаря:

```php
// Вычисляем статистику для каждого ножа
$statistics = [];
$todayStr = (new DateTime())->format('Y-m-d');

foreach ($knives as $knife_id => $knife) {
    $stats = [
        'in_stock' => 0,
        'in_sharpening' => 0,
        'in_work' => 0,
        'total_days' => 0,
        'last_status' => null,
        'last_status_date' => null
    ];
    
    // Подсчитываем дни в каждом статусе за период
    foreach ($dates as $date) {
        $status = $calendar_data[$knife_id][$date] ?? null;
        if ($status) {
            $stats[$status]++;
            $stats['total_days']++;
            // Обновляем последний статус (берем максимальную дату)
            if (!$stats['last_status_date'] || $date > $stats['last_status_date']) {
                $stats['last_status'] = $status;
                $stats['last_status_date'] = $date;
            }
        }
    }
    
    // Вычисляем проценты от общего количества дней в периоде
    $total = count($dates);
    $stats['percent_in_stock'] = $total > 0 ? round(($stats['in_stock'] / $total) * 100, 1) : 0;
    $stats['percent_in_sharpening'] = $total > 0 ? round(($stats['in_sharpening'] / $total) * 100, 1) : 0;
    $stats['percent_in_work'] = $total > 0 ? round(($stats['in_work'] / $total) * 100, 1) : 0;
    
    $statistics[$knife_id] = $stats;
}
```

## Особенности реализации

1. Статистика рассчитывается только для дней, где статус был установлен
2. Проценты рассчитываются от общего количества дней в периоде (3 месяца)
3. Последний статус определяется по максимальной дате установки
4. Адаптивная сетка - карточки автоматически подстраиваются под ширину экрана
5. Цветовая индикация соответствует цветам в календаре
6. Применяется к обоим файлам: knives_bobinorezka.php и knives_prosechnik.php
